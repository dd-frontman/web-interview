# Кэширование

Здесь собраны материалы по кэшу в вебе: зачем он нужен, где хранится и как влияет на скорость и нагрузку.

## Что важно понимать

- Кэш бывает на разных уровнях: браузер, CDN, сервер, приложение.
- Без стратегии инвалидции кэш быстро становится источником багов.
- Для собеседований важно уметь объяснить `Cache-Control`, `ETag`, `stale-while-revalidate` и практические сценарии.
