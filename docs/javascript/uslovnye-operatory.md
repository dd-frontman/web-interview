Вот набор условных конструкций / операторов, которые реально существуют в JavaScript и которыми чаще всего интересуются на фронтенд-собеседовании (Vue, React и т.д.).

| Что это                                                                                                                         | Когда используют                                                                                       | Краткий пример                                                                                                                                                            |
| ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`if … else` (+ `else if`)**                                                                                                   | Базовая ветвизация — выполнить один блок кода или другой                                               | `if (x > 0) { … } else { … }` [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/if...else)               |
| **Тернарный оператор `? :`**                                                                                                    | Краткая форма выбора значения прямо в выражении (часто в шаблонах Vue)                                 | `const msg = isOk ? '✅' : '❌';` [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Conditional_operator) |
| **`switch … case … default`**                                                                                                   | Когда вариантов больше двух и нужно сравнить одно выражение с несколькими значениями                   | ```js                                                                                                                                                                     |
| switch (status) {                                                                                                               |                                                                                                        |                                                                                                                                                                           |
| case 200: ok(); break;                                                                                                          |                                                                                                        |                                                                                                                                                                           |
| case 404: notFound(); break;                                                                                                    |                                                                                                        |                                                                                                                                                                           |
| default: error();                                                                                                               |                                                                                                        |                                                                                                                                                                           |
| }``` [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/switch) |                                                                                                        |                                                                                                                                                                           |
| \*\*Короткое логическое ветвление `&&` и `                                                                                      |                                                                                                        | `\*\*                                                                                                                                                                     |
| **Nullish coalescing `??`** оператор нулевого слияния                                                                           | Подставить значение-по-умолчанию **только** если слева `null` или `undefined`                          | `const port = cfg.port ?? 3000;` [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Nullish_coalescing)    |
| **Optional chaining `?.`** опциональная цепочка                                                                                 | Безопасно обратиться к вложенному полю; возвращает `undefined`, если что-то по пути — `null/undefined` | `user?.address?.city` [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Optional_chaining)                |

1. **`if/else` и тернарный** дают одинаковую логику; тернарный нужен, когда вы присваиваете/возвращаете значение, а не просто выполняете блок кода.
2. **`switch`** удобен, если «веток» много и проверяется одно выражение; не забывайте `break`, иначе случится «проваливание» (fall-through).
3. **`&&` и `||`** часто применяют в JSX/шаблонах Vue для условного рендера: `v-if="isReady && list.length"`.
4. **`??`** не путается с `||`: `||` смотрит на «falsy» (`0`, `''`, `false` тоже), `??` — только на `null`/`undefined`.
5. **`?.`** сокращает проверки «`obj && obj.prop`» и особенно ценится при работе с API-данными.
