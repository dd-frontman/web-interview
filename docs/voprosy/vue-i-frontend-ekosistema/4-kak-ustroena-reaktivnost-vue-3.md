# Как устроена реактивность во Vue 3 на высоком уровне?

## Ответ

Реактивность Vue 3 основана на системе отслеживания зависимостей.

Когда создаётся реактивный объект через `reactive` или `ref`, Vue оборачивает его в `Proxy`. `Proxy` перехватывает операции чтения и записи.

Когда компонент читает реактивное значение:

- Vue регистрирует зависимость;
- запоминает, что текущий эффект зависит от этого значения.

Когда значение изменяется:

- система реактивности уведомляет все эффекты;
- Vue запускает обновление компонента.

Основные элементы системы:

- `ref` — реактивное примитивное значение;
- `reactive` — реактивный объект;
- `computed` — вычисляемое значение;
- `watch` — реакция на изменение данных;
- `effect` — внутренний механизм реактивности.

Таким образом Vue обновляет только те части интерфейса, которые действительно зависят от изменившихся данных.
