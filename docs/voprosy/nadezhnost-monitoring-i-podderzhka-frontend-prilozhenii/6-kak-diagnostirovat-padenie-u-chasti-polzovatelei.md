# Как вы диагностируете проблему, если приложение падает только у части пользователей?

## Ответ

Такие проблемы обычно связаны с:

- конкретным браузером;
- определённой версией приложения;
- редким пользовательским сценарием;
- нестандартными данными.

В первую очередь анализируются данные из error tracking:

- какие браузеры затронуты;
- какие страницы;
- какие версии приложения.

Затем изучаются breadcrumbs и сетевые запросы.

Если проблема связана с данными, можно воспроизвести её на тестовом окружении с теми же входными параметрами.

Иногда приходится:

- анализировать network-логи;
- воспроизводить сценарий пользователя;
- запускать приложение в том же браузере.

Таким образом постепенно сужается область поиска причины.
