# Какие факторы вы уточняете перед тем, как предлагать стек для нового проекта?

## Ответ

Перед выбором технологий я стараюсь понять несколько вещей.

Во-первых, тип продукта:

- внутренний сервис;
- публичный веб-продукт;
- маркетинговый сайт;
- сложное SPA-приложение.

Во-вторых, требования к SEO. Если приложение должно индексироваться поисковыми системами, может понадобиться SSR или SSG.

В-третьих, нагрузка и масштабирование:

- сколько пользователей ожидается;
- насколько сложный интерфейс;
- есть ли real-time взаимодействие.

В-четвёртых, команда разработки:

- опыт команды;
- скорость разработки;
- требования к поддержке проекта.

В-пятых, инфраструктура:

- как происходит деплой;
- есть ли серверная часть;
- как будет организован CI/CD.

Выбор технологий должен исходить не из популярности библиотек, а из требований продукта и команды.
