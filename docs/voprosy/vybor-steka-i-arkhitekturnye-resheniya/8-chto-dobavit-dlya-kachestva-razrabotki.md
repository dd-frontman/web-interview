# Что бы вы добавили в проект помимо Vue, TypeScript и роутера, чтобы улучшить качество разработки и сопровождения?

## Ответ

Помимо основного стека важно добавить инструменты, которые помогают поддерживать проект в долгосрочной перспективе.

Во-первых, линтинг и форматирование:

- ESLint;
- Prettier.

Во-вторых, контроль качества коммитов:

- Husky;
- lint-staged;
- commit-lint.

В-третьих, тестирование:

- unit-тесты для бизнес-логики;
- e2e-тесты для ключевых пользовательских сценариев.

В-четвёртых, наблюдаемость системы:

- сбор ошибок;
- метрики производительности;
- мониторинг состояния приложения.

И ещё один важный момент — bundle-анализ. Инструменты анализа бандла помогают контролировать размер приложения и выявлять тяжёлые зависимости.

Такая инфраструктура позволяет поддерживать проект стабильным по мере его роста.
