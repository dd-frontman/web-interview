# Какие инструменты и библиотеки вы считаете базовыми для современного frontend-проекта?

## Ответ

Минимальный набор обычно включает несколько категорий инструментов.

Инструменты разработки:

- TypeScript;
- ESLint;
- Prettier;
- Vite.

Архитектурные библиотеки:

- Vue Router;
- Pinia.

Качество кода:

- unit-тесты;
- e2e-тесты;
- commit-hooks.

Инфраструктура:

- CI/CD;
- автоматическая проверка линтинга;
- автоматическая сборка.

Наблюдаемость:

- error tracking;
- метрики производительности;
- мониторинг ошибок.

Такой базовый набор позволяет поддерживать стабильность проекта и облегчает масштабирование команды.
