# Принципы программирования

Раздел про инженерные принципы, которые помогают писать предсказуемый, поддерживаемый и расширяемый код.

## Что внутри

- [SOLID](/printsipy-programmirovaniya/solid) — базовые принципы проектирования классов и модулей.
- [Маленькие принципы](/printsipy-programmirovaniya/malenkie-printsipy) — практические правила для повседневной разработки.

## Как применять

Используй эти материалы как чеклист при ревью и рефакторинге: меньше связности, понятные границы ответственности, проще тестировать.
